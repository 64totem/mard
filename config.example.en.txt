====== Totem ======
- English (Fa version: config.example.fa.txt)

To use Totem, you must configure the "config.json" file.
This file is for guidance only.

=== 1 === AppScript Key ======
Example: "appScriptKey": "AKfycbxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

Enter the AppScript Key that you created.

* To create an AppScript Key, follow the tutorial on the GitHub page or read the file "app-script-code.txt".

=== 2 === Batch Size ==========
Example: "batchSize": 5

* It's recommended not to change this.

Batch Size specifies how many segments are included in each download batch.

* Only used in the "download" type.

When downloading, Totem groups file segments into batches.
"batchSize" determines how many segments each batch contains.

=== 3 === Max Size ============
Example: "maxSize": 10

* It's recommended not to change this.

Some M3U8 playlists may contain a very large number of segments, which can reduce performance.

* Only used in the "stream" type.

=== 4 === Download Mode =======
Example: "downloadMode": "go"

* It's recommended not to change this.

Available modes:
-> "go"
-> "nr"

go - Downloads all segments simultaneously (recommended for faster internet connections).

nr - Downloads segments one by one in order (recommended for slower internet connections).

=== 5 === Low Latency ==========
Example: "lowLatency": false

Available values:
-> false
-> true

* Only used in the "stream" type.

true - Lower latency (recommended for faster internet connections).

false - Higher latency (recommended for slower internet connections).

=== 6 === Download Path =======
Example: "downloadPath": "downloads/"

* It's recommended not to change this.

"downloadPath" is NOT the output folder for the final MP4 file.

It is only used to store temporary ".ts" and ".m3u8" files.

=== 7 === Watch Port ==========
Example: "watchPort": 1819

* It's recommended not to change this.

* Only used in the "stream" and "watch" types.

=== 8 === FFmpeg Path =========
Example: "ffmpegPath": "ffmpeg.exe"

Enter the path to "ffmpeg.exe".

If you downloaded Totem from the GitHub Releases, you usually don't need to change this value.

If FFmpeg is already installed on your system and the "ffmpeg" command works, you can leave "ffmpegPath" empty ("").

If you are not using Windows, install FFmpeg manually and leave "ffmpegPath" empty ("").
