====== Totem ======
- Farsi (English version: config.example.en.txt)

Baraye estefade az Totem bayad file "config.json" ro tanzim konid.
In file faghat baraye rahnamaii ast.

=== 1 === appScript Key ======
Example: "appScriptKey": "AKfycbxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

Baraye tanzim, Bayad Appscript key ke sakhtid ro varded konid
- Baraye sakht AppScript Key bayad be amoozeshi ke dar safehe github ya file "app-script-code.txt" morajee konid

=== 2 === batch Size ========== 
Example: "batchSize": 5

- Dast nazanid behtare

Batch size yani meghdar har baste download.
- Faghat dar type "download" karbord darad.

Dar ghesmat Download Totem miad ghesmat (Segment) hay file ra dar baste (batch) gharar midahad.
batchSize meghdar segment haye dakhel har batch gharar migirad.

=== 3 === max Size =============
Example: "maxSize": 10

- Dast nazanid behtare

Momken ast dar file M3U8 meghdar ziadi az segment ha vojod dashte bashad va baes kondi shavad.
- Faghat dar type "stream" karbord darad.

=== 4 === download mode ========
Example: "downloadMode": "go"

- Dast nazanid behtare

Halat haye dar dastres
-> "go"
-> "nr"

go - Dar in halat download segment ha hame yek ja shorooe mishavad (baraye internet ghavi tar)
nr - Dar in halat download segment ha be tartib shorooe mishavad (baraye internet zaif tar)

=== 5 === low Latency ========
Example: "lowLatency": false

Halat haye dar dastres
-> false
-> true

- Faghat dar type "stream" karbord darad.

true - Takhir kamter (baraye internet ghavi tar)
false - Takhir bishtar (baraye internet zaif tar)

=== 6 === download Path ======
Example: "downloadPath": "downloads/"

- Dast nazanid behtare

In masiri ke downloadPath az shoma mikhahad be manai masir khorooji mp4 ha nist va marboot be
file haye .ts va .m3u8 ast.

=== 7 === watch Port ==========
Example: "watchPort": 1819

- Dast nazanid behtare

- Faghat dar type "stream" va "watch" karbord darad.

=== 8 === ffmpeg Path ==========
Example: "ffmpegPath": "ffmpeg.exe" 

Masir ffmpeg.exe ra vared konid.
Agar az release haye github Totem ro rikhtid niazi be taghir nist.

Agar ffmpeg dar system khod darid va fa command "ffmpeg" ejra mishavad mitavanid "ffmpegPath" ra khali "" begzarid.
Agar windows nistid bayad ffmpeg ra jodagoone berizid va "ffmpegPath" ra khali "" begzarid.
